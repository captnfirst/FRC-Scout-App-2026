\# 🚀 AGR Scouting \& Intelligence Command Center



\*\*FIRST Robotics Competition (FRC)\*\* takımları için geliştirilmiş, The Blue Alliance (TBA) ve Statbotics API'leri ile entegre çalışan, yapay zeka destekli strateji ve saha gözlem (scout) platformu.

\## 🌟 Öne Çıkan Özellikler



\- \*\*📱 Canlı Saha Gözlemi (Match Scouting):\*\* Mobil cihazlarla tam uyumlu, ergonomik veri giriş ekranı. Otonom rotası çizme, yakıt performansı ve defans kalitesi ölçümü.

\- \*\*🛠️ Pit Scouting İstasyonu:\*\* Robotların fiziksel özellikleri, yürüyen aksam detayları (Swerve/Tank vb.) ve sahada çekilen fotoğraflarının depolandığı galeri.

\- \*\*🧠 Dinamik AGR Score Algoritması:\*\* Admin panelinden ağırlıkları (Auto, Teleop, Climb, EPA) anlık olarak değiştirilebilen ve tüm turnuva sıralamasını belirlediğiniz stratejiye göre yeniden hesaplayan pick-list motoru.

\- \*\*🔮 Maç Simülatörü \& Strateji Masası:\*\* Statbotics'in (Kağıt Üzerindeki EPA) kazanma tahminleri ile ekibin sahadan topladığı gerçek verileri çarpıştıran modül. Drive Team için otomatik taktiksel uyarılar üretir (Örn: \*"Dikkat, Kırmızı ittifaktaki X robotu son maçında bozuldu!"\*).

\- \*\*🔒 Yetki Yönetimi:\*\* Admin ve standart Scout kullanıcıları için rol bazlı erişim kontrolü.



\## 💻 Kullanılan Teknolojiler



\- \*\*Backend:\*\* PHP (Custom MVC Architecture)

\- \*\*Database:\*\* MySQL

\- \*\*Frontend:\*\* HTML5, CSS3, JavaScript (Vanilla \& jQuery), Bootstrap 5, AdminLTE 3

\- \*\*Veri Görselleştirme:\*\* Chart.js

\- \*\*API Entegrasyonları:\*\* The Blue Alliance (TBA) API v3, Statbotics API
